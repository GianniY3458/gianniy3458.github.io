# Markdown 教程（完整可复制版 · 实用技巧加强）

本教程包含：  
1. 完整 Markdown 基础语法  
2. 全部示例源码 + 效果展示  
3. 大量提升可读性、美观度、专业度的技巧  
4. 全文只有一个代码块，可整体复制

----------------------------------------
# 1. 标题（Heading）
```
## 源码：
# 一级标题  
## 二级标题  
### 三级标题  
```

## 效果：
# 一级标题
## 二级标题
### 三级标题

----------------------------------------
# 2. 粗体 / 斜体 / 删除线

## 源码：
```
\*\*粗体\*\*  
\*斜体\*  
\~\~删除线\~\~  
\*\*\_粗体 + 斜体\_\*\*
```

## 效果：
**粗体**  
*斜体*  
~~删除线~~  
**_粗体 + 斜体_**

行尾加两个空格  下一行换行  

<br> 也能强制换行<br>适用所有平台

# 7. 链接（Links）

## 源码：
```
\[普通链接](https://example.com)  
\[带标题链接](https://example.com "提示")  

引用式：  
\[参考][ref]  
\[ref]: https://example.com
```

## 效果：
[普通链接](https://example.com)  
[带标题链接](https://example.com "提示")  
[参考][ref]

[ref]: https://example.com

# 8. 图片（Images）

## 源码：
```
\!\[示例图](https://picsum.photos/150)

控制大小：  
\<img src="https://picsum.photos/200" width="120">

居中：  
\<p align="center"><img src="https://picsum.photos/200" width="150"></p>
```

## 效果：
![示例图](https://picsum.photos/150)

<img src="https://picsum.photos/200" width="120">

<p align="center"><img src="https://picsum.photos/200" width="150"></p>

# 9. 表格（Table）

## 源码：
```
\| 名称 \| 数值 \| 备注 \|  
\|------\|------\|------\|  
\| A \| 10 \| 示例 \|  
\| B \| 20 \| 示例 \|

居中 / 右对齐：  
\| 左 \| 中 \| 右 \|  
\|:---\|:---:\|---:\|  
\| A \| B \| C \|
```

## 效果：
| 名称 | 数值 | 备注 |
|------|------|------|
| A | 10 | 示例 |
| B | 20 | 示例 |

| 左 | 中 | 右 |
|:---|:---:|---:|
| A | B | C |

<div style="flex:1;">右栏内容</div>
</div>
