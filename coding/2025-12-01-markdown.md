# Markdown 教程（完整可复制版 · 支持内部真实代码块）

本教程包含基础与实用 Markdown 语法，所有示例都包含“语法源码 + 渲染结果”。  
外层使用 ```markdown，内部示例使用 ~~~ 作为代码块围栏，避免冲突。

----------------------------------------
# 1. 标题（Heading）

## 源码：
~~~
# 一级标题
## 二级标题
### 三级标题
~~~

## 效果：
# 一级标题
## 二级标题
### 三级标题

----------------------------------------
# 2. 粗体 / 斜体 / 删除线

## 源码：
~~~
**粗体**
*斜体*
~~删除线~~
**_粗体 + 斜体_**
~~~

## 效果：
**粗体**  
*斜体*  
~~删除线~~  
**_粗体 + 斜体_**

# 7. 链接（Links）

## 源码：
~~~
[普通链接](https://example.com)

[带提示链接](https://example.com "提示")

~~~

## 效果：
[普通链接](https://example.com)  
[带提示链接](https://example.com "提示")  
[引用式链接][ref]

# 13. 数学公式（LaTeX）

> 你的渲染器必须支持 MathJax，否则不会显示。

## 源码：
~~~
这是行内公式：$a^2+b^2=c^2$

这是块级公式：
$$
\int_0^1 x^2 dx = 1/3
$$
~~~

## 效果：
这是行内公式：$a^2+b^2=c^2$

这是块级公式：
$$
\int_0^1 x^2 dx = 1/3
$$
